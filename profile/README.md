# Stormer
## Introduction
<img width="1584" height="396" alt="LinkedIn cover - 1" src="https://github.com/user-attachments/assets/5a34a27a-f4d7-4eeb-b1e6-f8af564230bb" />

Stormer is a knowledge management tool that can both leverage multiple LLM services and graph based UI.
I've decided to develop this project after finding out that I tempt to scroll all the way up and down to follow up the context inside the conversation with AI.

Stormer는 아이디어 브레인스토밍을 더 효율적으로 할 수 있게 도와주는 서비스입니다.
기존의 채팅형 UI로 AI와 함께 브레인스토밍할 때는 이전 메시지를 다시 확인하려면 위로 많이 스크롤해야 했고, 문맥을 이어가며 질문하기가 번거로웠습니다.
Stormer는 이 문제를 해결하기 위해 프로젝트 구조와 그래프 기반 UI를 제공합니다. 사용자는 아이디어를 노드 단위로 탐색하며 문맥을 유지한 채 사고를 확장할 수 있습니다.
